IL Tract Data Selector

This map shows census tracts in the state of Illinois. It also contains data for each tract from 2010, 2015, and 2020

To visualize data in a bar chart, first select a variable from the dropdown menu, then select tracts you are interested in seeing.

The bar chart will visualize the total count for that variable (for example, total population or number of people in poverty).

You can clean your tract selection by pressing the "clear selection" button. You can change the variable shown in the bar chart by selecting a new variable from the drop down menu.
